# Shopping-Trends-Analysis
Python-based Exploratory Data Analysis (EDA) on customer shopping trends to uncover purchasing behaviors influenced by genders, promo codes, subscription status, and customer loyalty.
# 🛍️ Shopping Trends Data Analysis Project

## 📌 Project Overview
This project performs a comprehensive Exploratory Data Analysis (EDA) on customer shopping trends data using Python. The primary goal is to understand purchasing behavior and uncover actionable insights by examining how different factors like promo code usage, subscription status, and gender influence spending and order frequency. Additionally, the analysis highlights customer loyalty by identifying the top frequent buyers.
📊 Key Features & Analysis
Data Cleaning & Preparation:** Structured raw customer datasets, handled formatting, and prepared variables for targeted analysis.
Behavioral Segmentation:** Analyzed the correlation between promo code usage and final purchase amounts.
Subscription & Gender Impact:** Explored how being a subscriber vs. non-subscriber impacts overall sales, segmented by demographic factors (Gender).
Customer Loyalty Tracking:** Identified and isolated the top 10 most frequent buyers to provide recommendations for targeted retention campaigns.
Data Visualization:** Built clean, informative charts using Python visualization libraries to present findings clearly.
Skills & Tools Used
Language:Python
Libraries:Pandas, NumPy, Seaborn, Matplotlib, Plotly (for interactive charts).
Techniques:Exploratory Data Analysis (EDA), Customer Segmentation, Data Visualization, Data Wrangling.
